# Ecommerce Management System
A Python and PostreSQL based Ecommerce Management System with full CURD operations.

## Feature 
- Python-PostgreSQL connectivity using psycopg2
- Automatic database and table creation
- Add, view, update and delete records
- Secure password handling
- Command-line based application

## Technologies Used
- Python
- PostgreSQL
- psycopg2
- SQL

## How to Run
1. Install PostgreSQL
2. Install required library:
   pip install psycopg2
3. Run the project:
   Ecommerce_Management.py
